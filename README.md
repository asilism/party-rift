# 🎉 보드게임 파티 (BoardGameParty)

보드/파티게임을 모아 함께 즐기는 컬렉션. 7인치 휴대기기 **가로 와이드** 화면 기준으로 만들었어요.

첫 번째 게임은 **사다리 게임** — 주사위를 굴려 먼저 골인하면 우승!

## 기술 스택

- React + Vite
- 한 기기에서 핫시트(번갈아) 플레이
- 캐릭터: 12지신 이모지 (추후 이미지 교체 가능)
- 사운드: Web Audio API로 생성(에셋 파일 불필요)

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드 → dist/
npm test         # 게임 로직 유닛 테스트
```

브라우저를 **가로 방향**으로 두고 즐기세요. (세로면 회전 안내가 표시됩니다.)

## 게임: 사다리 게임 규칙

- 인원(2~5명) 선택 → 각자 12지신 말 선택(중복 불가, 선택 순서 = 턴 순서)
- 자기 턴에 주사위(1~6)를 굴려 나온 만큼 이동
- 올라가는 발판(사다리)·내려가는 발판(미끄럼틀, 하강 폭 작게)
- 골 칸(30) 이상 도달 시 골인(초과 허용) → 먼저 골인하면 우승

## 폴더 구조

```
src/
├─ App.jsx              # 화면 흐름(로비 ↔ 게임)
├─ lobby/               # 게임 컬렉션 진입점
├─ shared/              # 공통(12지신 데이터, 주사위, 사운드)
└─ games/ladder/        # 사다리 게임
   ├─ board.config.js   # 설정값(칸 수/발판 등) — 향후 설정패널 연동
   ├─ engine.js         # 순수 게임 로직(테스트 대상)
   ├─ LadderGame.jsx    # 게임 진행/애니메이션
   ├─ Board.jsx         # 뱀형 보드 렌더링
   └─ PlayerSetup.jsx   # 인원/말 선택
```

설계/규칙 상세는 [`KEYPOINT.md`](./KEYPOINT.md), 작업계획은 [`PLAN.md`](./PLAN.md) 참고.

## 로드맵 (향후)

- 설정패널: 칸 수 / 승리 조건 / 주사위 / 인원 / 사운드 조정
- 이모지 → 커스텀 캐릭터 이미지
- 게임 추가(컬렉션 확장)
