# 🎉 보드게임 파티 (BoardGameParty)

여러 보드/파티게임을 한 곳에 모아 **한 태블릿에서 다같이** 즐기는 컬렉션.
7인치 휴대기기 **가로 와이드** 화면 기준으로 만들었고, 어린이(6세+)도 쉽게 즐길 수 있게 직관적인 조작과 멀티터치를 지원합니다.

> 캐릭터는 **12지신 이모지**로 표현합니다. 참가자는 로비에서 한 번 정해두면 모든 게임이 공유해요.

| 💣 **스릴팡** | 폭탄이 블랙홀에 빠지기 직전, 아슬아슬하게 내 구슬 떨어뜨리기 | 2~4 | 타이밍 |

## 기술 스택

- **React + Vite** (의존성 최소)
- 한 기기에서 **핫시트 + 멀티터치**(여러 명이 동시에 자기 영역 터치) — `onPointerDown` 사용
- 캐릭터: **12지신 이모지** (추후 이미지 교체 가능)
- 사운드: **Web Audio API**로 생성 (에셋 파일 불필요)
- 참가자 명단은 **localStorage**에 저장(오프라인에서 새로고침해도 유지)
- 게임 로직은 **순수 함수(engine.js) + 유닛 테스트**로 화면과 분리

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드 → dist/
npm run preview  # 빌드 결과 미리보기
npm test         # 게임 로직 유닛 테스트(node --test)
```

브라우저를 **가로 방향**으로 두고 즐기세요(세로면 회전 안내 표시). 우상단 **⛶ 전체화면** 버튼 권장.

### 같은 네트워크/외부 기기에서 접속
`vite.config.js`에 `server.host: true`, `allowedHosts: true`가 설정돼 있어, `npm run dev` 실행 시 콘솔의 `Network: http://<내IP>:5173/` 로 같은 와이파이의 태블릿에서 바로 접속할 수 있습니다. (방화벽에서 Node 인바운드 허용 필요)

## 플레이 흐름

1. **로비**에서 참가자 추가(이름 + 12지신 말, 중복 불가) — 인원 미달 게임은 비활성.
2. 게임 선택 → 게임별 설정(보드 크기/난이도 등) → (해당 시) **차례 정하기**(주사위) → 플레이.
3. 우승 시 폭죽 연출 → 다시하기 / 로비로.

## 폴더 구조

```
src/
├─ App.jsx                 # 화면 흐름(로비 ↔ 게임) + 오프라인 저장
├─ lobby/GameLobby.jsx     # 컬렉션 진입점 + 참가자 관리 + 인원 게이팅
├─ shared/                 # 공통 모듈
│  ├─ zodiac.js            # 12지신 데이터
│  ├─ Dice.jsx             # 주사위(굴림 연출)
│  ├─ sound.js             # Web Audio 효과음
│  ├─ storage.js           # 참가자 localStorage 저장/복원
│  ├─ Fireworks.jsx        # 우승 폭죽
│  ├─ ShootingStars.jsx    # 콤보 별똥별
│  ├─ FullscreenButton.jsx # 전체화면 토글
│  ├─ TurnOrderRoll.jsx    # 차례 정하기(공용)
│  └─ ErrorBoundary.jsx    # 게임 오류 표시
└─ games/                  # 게임별 (각 폴더: engine.js + *.test.js + UI)
   ├─ ladder/  메모리 memory/  dobble/  thrillpang/  race/  whack/  traffic/
```

각 게임은 **`engine.js`(순수 로직)** 와 **`engine.test.js`(테스트)**, 그리고 화면 컴포넌트로 구성됩니다.
