# 보스전 심화 — 기믹 + 전리품 + 세트 효과 설계

작성: 2026-07-23. 상태: **4보스 주력 기믹 + 전리품 12종 + 세트 효과 4종 전부 구현 완료(v53~v55).**
- v53 그림자(gaze) 세로 슬라이스 → v55 나머지 3보스 확장: 거인=발구르기(용암 웅덩이+넉백),
  대마도사=성좌 낙인(STACK 피해 분할), 가시=반사창+낙인(기존 킷을 주력으로 공인)
- 남은 백로그: **P3 트위스트 + 소프트 인레이지**(공통 레이어) — 아래 설계 그대로 미착수

기믹 튜닝 교훈(다음 확장 때 필독):
1. **봇은 기믹에 최소 시간만** — gaze 채널 내내 멈추면 70%→30% 폭락. 집행 직전 0.9초만 등돌림으로 복원
2. **기믹은 한 번에 하나 + 직후 한숨** — STACK 진행·직후에 다른 장판을 겹치면 0/16 전멸.
   낙인 동안+폭발 후 2.4초 다른 기술 홀드로 38%(베이스라인 동률) 복원
3. **잔류 장판은 봇이 밟고 서 있으면 걸어 나오게** — 발구르기 웅덩이에서 44%→25% 출혈,
   botDodgeBossZone에 잔류 탈출 추가로 38% 복원
4. STACK 튜닝 최종: 총피해 = 대상 최대체력 ×1.45, 쿨 30초(2.0·21초는 전멸, 1.6·24초도 부족)
목적: 보스전이 "증강 없는 디펜스"처럼 얇게 느껴지는 문제 해소 — 두 모드의 정체성을 갈라 겹침을 없앤다.

## 배경 / 핵심 방향

디펜스에 강화·증강이 붙으며 "성장의 재미"를 독점 → 보스전이 상대적으로 볼륨이 얇아 보임.
두 모드가 "5명이 강한 적을 잡는다"는 같은 축에서 경쟁하는 게 근본 원인.

> **디펜스 = "강해진다"** (빌드·물욕·끝없는 성장 — get stronger)
> **보스전 = "능숙해진다"** (패턴·실행·숙련 — get better)

보스전을 **성장이 아니라 실행/숙련** 축으로 밀어 겹침을 없앤다. 공유 보스 로스터는 서사로 정당화:
**보스전 = 그 괴물의 "진짜 형태"(풀 기믹+인레이지+전리품), 디펜스 보스 = 파도에 섞인 "메아리"(브루트판, 현행).**

⚠️ 보스전에 증강을 넣지 말 것 — 겹침이 오히려 심해진다. 보스전은 "빌드가 아니라 실력"으로 간다.

# 1. 보스별 시그니처 기믹

레이드 어휘(와우/FF14) 중 톱다운·모바일·봇 아군·5인에 맞는 것만 선별. 각 보스는 서로 **다른 근육**을 쓴다.
기존 3페이즈(70%/40% 체력)에 **주력 기믹 + 40%↓ 트위스트**를 얹어 진행할수록 심화.

## 👹 파멸의 거인 (대지/힘, lava) — 회피·넉백 (읽기 쉬운 힘)
"무식한 힘" 테마엔 협동보다 직관적 회피가 맞다(캐주얼 유저 첫 보스).
- **주력**: 대지 내려찍기 — 예고 원에서 **빠져나가기**(원 안 즉사급). 발구르기 **넉백** → 뒤의 위험지대(가시밭/균열) 회피
- **P3 트위스트**: 발구르기 **충격파 링(도넛)** — 안전지대는 보스 품 안, 링은 넘어야
- *구현*: pushBossZone(원/도넛) 재사용 + 넉백(windcaller push 재사용). 봇=예고 회피(botDodgeBossZone 이미 됨)

## 🧙 대마도사 (운석·얼음/마법, frost) — STACK + SPREAD (마법 광역)
마법사가 광역을 까는 게 자연스러워 stack/spread 짝을 여기 배치. stack을 "가장 헷갈리는 기믹"으로 두되 첫 보스가 아닌 두 번째라 학습 곡선 OK.
- **주력(STACK)**: 대상 1명에 거대 광역 조준, **총 피해를 원 안 인원수로 분할** → 다 모여 나눠 맞기
- **P3 트위스트(SPREAD)**: 다중 운석 표식, **표식끼리 가까우면 폭발 겹쳐 치명** → 산개
- *구현*: 피해 분할(신규, damageHero 분할) + 표식/조준낙하. 봇=집결/산개 행동 신규

### STACK 튜닝 원리 (중요 — 헷갈리기 쉬움)
- **기하급수 아님 = 단순 나눗셈(반비례)**: 각자 = 총피해 ÷ 인원. 4000이면 1명 4000/2명 2000/3명 1333/4명 1000/5명 800. **감소폭이 앞에서 최대**(1→2 절반), 뒤는 완만 → "한 명이라도 빠지면 확 아파짐"이 긴장감
- **체력에서 거꾸로 튜닝**: 혼자=체력 150~200%(확실한 즉사, 모여야 하는 이유) / 풀집합=체력 30~50%(무섭지만 생존, 안 아프면 무의미). 예: 물몸 1800체력 → 풀집합 목표 40%≈720 → 총피해 720×5=3600 → 혼자 3600=200% 즉사

### STACK/SPREAD 표시 (FF14 벤치마크 — 헷갈림 방지 3요소)
1. **화살표 방향**: Stack=안쪽 모임 `→⦿←`("나한테 모여") / Spread=바깥 뻗음 `←⦿→`(개인 폭탄, "떨어져")
2. **색**: Stack=파랑/흰(안전한 모임) / Spread=주황/빨강(개인 폭탄)
3. **바닥 예고 원 + 줄어드는 카운트다운 링**(타이밍). 5인·작은 폰 화면 → **대상 머리 위 큰 아이콘 필수**, 색+화살표로 0.5초 안에 "모여?/흩어져?" 판단

## 👺 그림자 군주 (어둠·암살·공포, shadow) — GAZE (등 돌리기) ← 시그니처
가장 독창적. 이 게임의 대표 기믹이 될 잠재력.
- **주력**: 공포의 응시 — 시전 중 **보스를 바라보는 영웅만 공포**, 등 돌리면 회피(메두사식)
- **P3 트위스트**: 은신 습격 처형 — 사라졌다 최저체력 영웅 뒤에서 급습, 돌아서 마주보면 반감
- *구현*: fear + 영웅 dir vs 보스 방향 판정(신규). 봇=시전 감지 시 반대 방향 조준(신규)

## 🌵 가시군주 (자연·가시·반사, venom) — STOP-ATTACK + SPREAD ← 이미 절반 존재
- **주력(STOP)**: 반사창 — 반사 중 때리면 되받음 → **공격 중단**(인내 체크). 낙인 폭발 → **산개**
- **P3 트위스트**: 가시밭 도넛(안전=품 안) + 낙인 가속
- *구현*: thornArmor/thornBomb 이미 존재 → 정식화 + 도넛 추가

→ **빠져 / 모여·흩어져 / 등 돌려 / 멈춰** — 네 보스가 완전히 다른 판단. 3개는 기존 시스템 재사용, gaze·stack만 신규.

## 공통 레이어 — 소프트 인레이지 (여유되면)
제한시간 초과 시 보스 폭주(사실상 클리어 불가) → "패턴 배워 시간 안에 잡는" DPS·실행 레이스.
디펜스의 "무한 버티기"와 정반대 축 → 겹침 해소의 결정타.

## 난이도 스케일 & 봇
- **쉬움**: 예고 길게, 실패해도 안 죽을 만큼(학습) · **악몽**: 예고 짧게 + 주력 치명 · **지옥**: P3 트위스트 풀가동 + 인레이지 빡세게
- **봇은 기믹마다 전용 행동 필요**(집결/산개/등돌리기/멈춤/회피) — 예고 회피는 됨. 이게 기믹당 숨은 구현 비용

# 2. 보스별 전리품 세트 (꾸미기, 비매품)

보스마다 테마 **3피스**, **난이도로 잠금 해제**: 쉬움→🎩모자, 악몽→🧣옷, 지옥→🗡️무기.
**3난이도 다 깨면 그 보스 세트 완성**(숙련 목표).

| 보스 | 🎩 모자(쉬움) | 🧣 옷(악몽) | 🗡️ 무기(지옥) |
|---|---|---|---|
| 파멸의 거인 (용암) | 용암 뿔투구 | 마그마 갑주 | 대지파쇄 망치 |
| 대마도사 (운석·얼음) | 성운의 관 | 은하 로브 | 운석 지팡이 |
| 그림자 군주 (어둠) | 그림자 가면 | 심연 망토 | 그믐의 낫 |
| 가시군주 (자연) | 가시 왕관 | 덩굴 갑옷 | 가시 대검 |

총 **12종**. ⚠️ **최대 작업량 = scene.js 절차적 3D 모델 12개 신규**. 데이터/UI/해금보다 모델링이 병목.

# 3. 비매품 표시 (구매 불가 구분)

- 데이터: 꾸미기 항목에 `trophy: { boss, tier }` 추가(가격 없음). 예: `{ id:'colossus_hat', name:'용암 뿔투구', trophy:{ boss:'boss_colossus', tier:'normal' } }`
- HatScreen(SoloApp): trophy 항목이면
  - **미해금**: 자물쇠 + 코인 가격 자리에 "🏆 파멸의 거인 · 지옥 토벌" 조건 표시, 구매 버튼 없음
  - **해금(보유)**: 금테 + "전리품" 리본 → 일반 판매템과 확실히 구분, 장착 가능
- 저장/해금: `recordBossClear`(SoloApp onFinish 승리 처리) 시 해당 tier 전리품을 `addOwnedHat/Costume/Weapon`으로 지급
- WARDROBE_TABS·HATS/COSTUMES/WEAPONS 구조 그대로 확장 + `trophy` 필드 분기

# 4. 세트 효과 (A안 — 풀세트만·PvE 전용·소효과)

**개별 조각은 스탯 없음(순수 꾸미기).** 한 보스 **풀세트(3개) 완성 시에만** 작은 테마 효과 + 전용 오라.
- 거인 세트 = 공격력(또는 체력) +2~3% / 대마도사 = 주문력 +2~3% / 그림자 = 이동속도 +? / 가시 = 피해감소 +?
- ⚠️ **콜로세움(PvP)에선 전부 비활성** — 공들여 맞춘 밸런스 보호(전리품만 있는 사람이 유리해지는 grind-to-win 방지). PvE(보스전·무한방어)에서만 적용
- 세트 완성 시 테마 색 **오라**(명예 표시) — 스탯보다 이게 진짜 보상감
- 구현: 착용 꾸미기 3개가 한 보스 세트면 보너스 on(mode !== 'arena' 게이트). itemBonus/heroAtk 등 기존 스탯 파이프라인에 소량 가산

# 구현 스코프 & 순서

**병목 = 꾸미기 3D 모델 12개(scene.js 절차 빌더).** 나머지(데이터·UI·해금·효과 게이트)는 가벼움.

### 세로 슬라이스 먼저 — 보스 1종(예: 그림자 군주)으로 전 흐름 검증 ✅ 완료(v53)
① gaze 기믹(엔진+봇) → ② 3피스 전리품 모델 + trophy 데이터 → ③ 비매품 UI → ④ 클리어 해금 훅 → ⑤ 세트 효과(PvE) → ⑥ 실기기 확인. 흐름 확정 후 나머지 3보스 확장.

구현 위치(다음 보스 확장 시 그대로 따라가면 된다):
- 기믹: engine.js `GAZE_*` 상수 + bossShadow 시전 + bossThink 펜딩 집행 + `botGazeAvert`(집행 직전만 등돌림) + makeView `bossGazeT`
- 연출: scene.js 보스 가슴께 👁️(점점 커짐) + 영웅 머리 위 👁️ 경고(바라보는 동안만 — 등 돌리면 즉시 꺼져 회피 성공이 읽힌다)
- 전리품: SoloApp `HATS/COSTUMES/WEAPONS`에 `trophy:{boss,tier}` 항목 + `grantBossTrophies`(onFinish) + HatScreen 잠금/전리품 분기
- 모델: scene.js `shadowmask`(HAT_BUILDERS/FX) · `abysscloak`(COSTUME_BUILDERS/FX) · `crescentscythe`(WEAPON_SKINS/FX) — 은빛 그믐달 문양으로 세트감 통일
- 세트 효과: engine.js `TROPHY_SETS`/`trophySetOf` + roster.js에서 장착 판정 → netgame/makeHeroState 전달 → createGame이 `isRaidMode` 아니면 제거(콜로세움 누수 원천 차단) + scene `buildTrophyAura`(발밑 보라 링·입자)

### 이후
- 나머지 3보스 기믹 + 전리품(모델 9개)
- P3 트위스트 + 소프트 인레이지(공통 레이어)

### 엔진/코드 훅 참고
- 예고 장판: `pushBossZone`(원/도넛/선) 재사용 / 넉백: windcaller push 재사용 / 마크: thornBomb 재사용 / 반사: thornArmor 존재
- 신규: 피해 분할(stack, damageHero), 응시 방향판정(gaze), 봇 집결/산개/등돌리기 행동
- 보스 페이즈(`bossPhase` 70/40%)에 트위스트 부착 / 전리품 해금은 `recordBossClear`
- 꾸미기: `WARDROBE_TABS` + `HATS/COSTUMES/WEAPONS`(SoloApp) + scene.js 빌더
