# 보스전 공격 패턴 정리 (v2 — 3역할 체계)

> 이 문서가 곧 스펙이다. 모든 패턴은 "보이는 것 → 일어나는 일 → 사는 법" 3줄로 정리한다.
> v1의 정리 포인트 A~F는 전부 반영됨(하단 이력 참조).

---

## 0. 설계 원칙 (v2에서 확립)

1. **역할 분리**: 건물 철거는 병사의 일(진군 후 건물 피해 100%), 영웅 사냥은 보스의 일(보스 건물 피해 10%). 병사는 영웅에겐 잽 수준(55%) — 파도를 치우는 게 수성이자 파밍이고, 그동안 보스가 우리를 사냥하러 온다.
2. **파도 문법 통일**: 다단 패턴은 반드시 **안쪽부터 터진다** — "이미 터진 자리는 안전하다". 도넛 단일 폭발(파고들기 강요)은 폐기.
3. **3역할 킷**: 보스당 스킬은 정확히 3개 — ① 읽기 패턴(크고 느리고 치명) ② 압박기(중형 광역) ③ 처형기(표적 지정 + 짧은 예고).
4. **예고 시간 규칙**: 피해×범위가 클수록 예고가 길다. 처형기 0.6~0.75초 < 압박기 0.9~1.2초 < 읽기 패턴 1.2~2.4초(다단). 예고 0초 기술은 없다.
5. **사거리 역전**: 보스 평타(13.2~15.5, +국면당 15%) > 아군 최장(12.5) > 타워(13) — 포킹은 없다. 싸움은 우리가 건다.
6. **부활 상한 18초**(아군) — 죽음은 리셋이 아니라 잠깐의 공백.

## 0-1. 예고를 읽는 법

| 신호 | 의미 |
|---|---|
| 색 링이 조여들며 짙어짐 | 그 자리에 폭발 — 다 조여들면 터진다 |
| 여러 겹 링이 겹쳐 보임 | 파도 — **안쪽부터 차례로** 터진다. 터진 자리를 밟아라 |
| 색 | 🟠 카르곤(용암) · 🔵 아르케인(서리) · 🟣 녹스(어둠) |
| 폭발 후 바닥에 색이 남음 | 잔류 장판(용암/서리밭) — 밟으면 도트+둔화 |
| 내 발밑의 작은 표식 | 처형기 조준 — 즉시 그 자리를 떠나라(연쇄 표식은 팀에서 떨어져라) |
| 보스 발밑 위협 링 | 국면: 🔴 진군 → 🟠 분노(70%) → 🟣 필사(40%, 맥동) |
| 보스가 보라 구체에 감싸임 | 각성 휴지기 30초 무적 — 때리지 말고 정비·파밍 |

## 1. 공통 시스템

| 패턴 | 보이는 것 | 일어나는 일 | 사는 법 |
|---|---|---|---|
| 어둠 탄환 (평타) | 보스가 나를 바라봄 | 사거리 13.2~15.5 원거리 타격 | 못 피함 — 우리가 붙어야 보스가 건물 대신 우리를 본다 |
| 전환 충격파 (국면 전환) | 보스 중심 r11 링, 예고 1.2초 | 폭발 + 기절 0.8초 | 링 밖으로 |
| 병사 파도 | 병사 무리 솟음 | ~2:30 14마리/10초 · 진군 후 14마리/13초 — **건물 피해 100%(진군 후)** | 전선에서 요격 — 방치하면 타워가 진짜 무너진다. 골드 농장이기도 |
| 그림자 영웅 (2:30, 40%) | 부채꼴 5기 등장 | 아군 평균 레벨 정예, 후퇴·부활 없음, 건물 무관심 | 각개 처치 — 킬 골드가 연료 |
| 각성 휴지기 (70%/40%) | 보라 보호막 구체 30초 | 무적·정지(소환·성장도 정지) | 정비/보급/파밍 — 종료 6초 전 재집결 |
| 광폭화 (진군 후 11분) | 피드 + 붉은 이펙트 | CC 무시 + 피해 1.5배 | 전멸 타이머 |

## 2. 👹 카르곤 — 파멸의 거인 (용암 🟠) — "밟는 자리를 읽어라"

| 역할 | 스킬 (쿨) | 보이는 것 | 일어나는 일 | 사는 법 |
|---|---|---|---|---|
| 읽기 | **단층선** (18초) | 표적 방향 부챗살 3~5갈래로 주황 링 줄지어 깔림 | 안쪽부터 물결 폭발 + 기절 0.5 | 갈래 **사이 틈**에 서기 |
| 압박 | **대지 강타** (9초) | 발밑 주황 링 r9~, 예고 1.1초 | 광역+기절 1초. 분노: 용암 잔류+파생 균열 / 필사: 3중 파문(안→밖) | 링 밖으로. 파문은 터진 안쪽 밟기 |
| 처형 | **격돌 돌진** (16초) | 내 발밑 r5 착지 표식, 예고 0.7초 | 그 지점으로 돌진 + 폭발 기절 1초 | 표식에서 **걸어 나가면 빗나간다** |
| 전용 | **파멸의 삼중격** (6초) | 보스가 표적을 향해 몸을 틂 | 전방 3갈래 검기 — 근거리 겹침 | 정면·근거리 금지, 옆으로 반 발짝 |

|---|---|---|---|---|

## 5. 밸런스 스냅샷 (봇 5인 시뮬, 각 8판)

카르곤 6/8 · 아르케인 5/8 · 녹스 5/8, 평균 7~8.5분.
**주의**: 봇은 예고를 완벽 회피하는 모범생 — 이 밴드는 "공략을 아는 팀"의 상한이다.
학습 전의 사람은 이보다 어렵고, 공략을 익힐수록 여기에 수렴한다(의도된 곡선).

## 6. 반영 이력

- v1 진단 A(도넛 직관성) → **파도 문법 통일**로 해소: 빙하기 → 밀려오는 한파(3겹 안→밖). 어둠의 파문은 원래 이 문법
- B(패턴 과다) → 3역할 킷: 회전 격노·멸망의 운석·어둠걸음 삭제
- C(삼중격 공통) → 카르곤 전용화
- D(예고 위계) → 돌진 0.7초 착지 표식 / 뇌격 0.6초 ⚡ 표식 — 예고 0초 기술 소멸
- E(회피 불가 비율) → 남은 회피 불가 딜은 평타뿐(자리 유지 비용 역할)
- F(트리거 이원화) → 자기 중심 스킬=품 안 스캔 / 표적 스킬=foe, 예고 집행은 bossDash/bossChain 펜딩으로 일원화
- \+ 역할 분리(건물=병사·영웅=보스), 부활 상한 18초
