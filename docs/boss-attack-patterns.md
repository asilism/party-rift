## 5. 정리 포인트 톺아보기 (진단)

### A. 도넛 "안쪽 안전지대"가 직관과 싸운다 ← **가장 큰 문제 (유저 지적)**
- 위험이 다가오면 **멀어지는 게** 본능인데, 빙하기·어둠의 파문은 "보스 품으로 파고들라"고 요구한다. 초록 경계선을 그려도 학습 전엔 읽히지 않고, 학습 후에도 "왜?"라는 서사가 없다(보스 품이 왜 안전한가?).
- **제안 1 (교체)**: 도넛 폐기 → **"밀려오는 파도"**로 재해석. 보스 중심에서 바깥으로 링이 *순차적으로* 터져 나감(현 어둠의 파문의 반대 서사) — "파도가 밀려온다 → 파도를 등지고 달리면 죽는다 → **터진 자리를 밟고 서면 산다**". 밖으로 뛰지 말고 제자리 근처에서 "이미 터진 안쪽 밴드"로 반 발짝 — 직관(폭발 자리는 이제 안전)과 일치.
- **제안 2 (교체)**: 안전지대를 **초록 원판으로 직접 칠하기** — "여기로 모여!" 개념. 폭발 전 무작위 위치 2~3곳에 초록 원이 뜨고, 나머지 전장이 위험. 직관적("초록 = 안전")이고 팀이 모이는 그림도 예쁘다.
- **제안 3 (유지+보강)**: 도넛을 유지하되 서사를 입힘 — 아르케인이 "폭풍의 눈"을 시전(태풍 비주얼: 바깥은 눈보라 소용돌이, 중심은 맑음). "태풍의 눈은 고요하다"는 보편 상식으로 직관을 보완.
- 추천: **제안 1** (카르곤 3중 파문과 문법 통일 — "안쪽부터 터진다"를 전 보스 공통 규칙으로).

### B. 패턴 과다 — 한 보스가 기술 6종을 돌린다
- 평타 + a/b/c + 시그니처(d) + 삼중격(fan) = 장판이 겹쳐 화면이 읽히지 않는 순간이 생긴다(특히 필사 국면: 융단+운석+빙하기+삼중격 동시).
- **제안**: 보스당 역할을 3개로 슬림화 — ① **시그니처 읽기 패턴**(단층선/빙하기류/파문 — 크고 느리고 치명) ② **압박기**(강타/폭풍/포효 — 중간) ③ **처형기**(돌진/운석/습격 — 표적 지정). 삼중격은 공통기에서 빼고 **카르곤 전용**으로(전사형의 정체성), 아르케인은 뇌격을, 녹스는 습격을 처형기로. 회전 격노·어둠걸음은 국면 연출로 흡수하거나 삭제 후보.

### C. 삼중격이 세 보스 공통 → 개성 희석 + 비주얼 불일치
- 검기(검성 이펙트 재사용)가 마법사·암살자와 안 어울린다. B안대로 카르곤 전용화하거나, 보스별 투사체(용암 파편/얼음 창/그림자 표창)로 스킨 분리.

### D. 예고 시간 위계가 뒤죽박죽
- 현재: 습격 0.75 < 포효 0.9 < 강타 1.1 < 충격파·폭풍 1.2 < 단층선 1.3+ < 운석 1.4 < 빙하기 2.4. 그런데 **격돌 돌진·연쇄 뇌격은 예고 0초**인데 피해는 중상급.
- **제안**: 규칙 명문화 — "피해 × 범위가 클수록 예고가 길다". 즉발은 (i) 약하거나 (ii) 회피 대신 다른 카운터(산개/차단)가 있어야. 돌진에 0.6초 조준선 예고, 뇌격은 첫 표적에게 ⚡ 마크 0.5초 예고 추가 검토.

### E. 회피 불가 딜의 비율 원칙 부재
- 원칙 제안: 보스가 주는 피해의 **70% 이상은 "예고를 보고 피할 수 있는" 기술**에서 나와야 한다. 평타·뇌격 같은 회피 불가 딜은 "자리 유지 비용"(지속 압박) 역할까지만.

### F. 트리거 조건 이원화 (코드 위생)
- 사거리 역전 후 자기 중심 스킬은 "품 안의 적 스캔", 표적 지정 스킬은 "foe 거리"로 나뉘어 있다 — 동작은 맞지만 규칙이 코드에 흩어져 있음. 스킬 정의를 데이터화(트리거 종류·예고·범위·효과를 한 테이블로)하면 B·D 정리와 함께 자연히 정돈된다.

### G. 잘 되어 있는 것 (유지)
- 색 = 보스 속성 언어(주황/청/보라), 국면 = 위협 링·체력바 마커·몸집, 각성 휴지기의 완급, "예고 → 폭발 → 잔류"의 3박자, 사거리 역전("우리가 먼저 건다"), 미니맵 비컨.
