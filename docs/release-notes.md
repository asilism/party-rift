# 출시 노트 (Play Console 붙여넣기용)

각 언어 500자 제한. 업로드하는 트랙의 "버전 세부정보 > 출시 노트"에 붙여넣기.

## v1.1 (versionCode 25) — 2026-07-19

### ko-KR

```
🏟️ 대규모 콘텐츠 업데이트!
• 신규 모드 2종 — 무한 방어(끝없는 파도에서 수호석 사수), 콜로세움(12지신 2:2 서바이벌 토너먼트, 매 라운드 무작위 경기장)
• 보스전 난이도 3단계(쉬움·악몽·지옥) + 신규 보스 "브램블" 추가
• 업적 40종 + 칭호 수집 — 달성하면 이름 옆에 칭호 표시
• 캐릭터 선택창 개편 — 스킬을 탭하면 자세한 설명
• 상점 버튼 위치 개선(미니맵 아래), 모든 화면 비율 대응

• 귀환 후 캐릭터가 저절로 걷던 문제 수정, 발열·성능 개선
```
